# 367 有效的完全平方数

**题目：**

给定一个正整数num，编写一个函数，如果num是一个完全平方数，则返回true，否则返回false。

进阶：不要使用任何内置的库函数，如sqrt。



**代码实现：**

```java
class Solution {
    public boolean isPerfectSquare(int num) {
        int left = 0, right = num;
        while (left <= right) {
            int mid = (right - left) / 2 + left;
            long square = (long) mid * mid;
            if (square < num) {
                left = mid + 1;
            } else if (square > num) {
                right = mid - 1;
            } else {
                return true;
            }
        }
        return false;
    }
}
```
