# 242 有效的字母异位词

**题目：**

给定两个字符串s和t，编写一个函数来判断t是否是s的字母异位词。



**代码实现：**

```java
class Solution {
    public boolean isAnagram(String s, String t) {
        int[] record = new int[26];
        for(char c : s.toCharArray()) {
            record[c - 'a'] += = 1;
        }
        for(char c : t.toCharArray()) {
            record[c - 'a'] -= 1;
        }
        for(int i : record) {
            if"(i != 0) {
                return false;
        }
        return true;
    }
}
```
